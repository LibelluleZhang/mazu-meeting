妈祖 · 剧本与角色

创作圣经 · 2026.06.22
以林默出生到羽化的 10个核心传说 + 焚屋引航 为主要素材，扩写改编为 24集 × 15分钟 的单元剧。 三集一个故事单元，每个单元让林默获得新认识——对世界的认识、对人的认识，神通逐渐增强。 感情线以亲情为核心，不设爱情线。反派 海燕公 贯穿全剧。 开篇双版本报批：郑成功海战版 / 阿拉伯海战版，由协会审核决定。
故事素材 · 10传 + 焚屋引航
0 1
妈祖诞降
深夜出生，满屋红光。从出生到满月不哭不闹，取名「默」。一个家庭如何接纳一个不一样的孩子。
0 2
窥井得符
16岁，井水中浮现神人捧铜符。女伴全跑了，她独自伸手接住。从此能预知吉凶。
0 3
机上救亲
织布机前神魂出游，海上救父兄。母亲将她摇醒。父亲得救，哥哥坠海。第一次救人，只救回一半。
0 4
化草救商
商船触礁将沉。她拔一把草抛向海面。草化为杉木，托住船身。船员靠岸才知是神力。
0 5
挂席泛槎
船夫因风浪不敢出海。她挂草席当帆，小船如飞鸟掠海。草席浸水不湿。
0 6
铁马渡江
渔船全出海。她取下屋檐铁马，化为青白骏马，踏浪而去。回来后铁马不再响了。
0 7
祷雨济民
莆田大旱。她登山顶设坛，说申时下雨。午时烈日，无人信。申时暴雨倾盆。
0 8
降伏二神
收服顺风耳与千里眼。三战三收，敌人化为护法。从此站在妈祖身后。
0 9
焚屋引航
夜黑风高，船队迷航。她点燃自家屋顶为海面引路。船队归来，家化灰烬。父亲问值不值得——她没有回答。
1 0
湄屿飞升
28岁，独自走上山顶。云气遮山，钟鼓丝竹。人间少女转入神格。
剧情结构
单元剧形式。每个小单元让林默获得新认识，神通逐渐增强。故事前期积累能力（出生→窥井得符→治病救人），中期节奏加快（化草救商→铁马渡江→遭遇更大挑战），情感冲突集中爆发（机上救亲→与母亲决裂→陷入低谷），重拾初心（降伏二神→主动帮助民众→祈雨→焚屋），最终抉择（海难/海战→羽化飞升）。

龙凤喜灯
明 线
初始为简单灯，林默有法力后可用意念控制开关。象征引航。贯穿全剧，从出生时的灯到飞升时照彻海面。
⚓
船锚
暗 线
在海难或引航时暗戳戳出现。象征稳定。不喧哗，但每一次风暴中都在——像妈祖自己。
感情线 · 亲情为核心

兄妹情

哥哥代表保护。在母亲期望林默嫁人时，哥哥是家庭矛盾的润滑剂。机上救亲后哥哥遇难，林默陷入自责——「如果我当时出海了，是不是两个都能救。」这是她人生的低谷。哥哥的死在后续情节中反复被唤起，成为她救人的动力，也是她与母亲之间最深的裂痕。

母女情

母亲代表世俗期望——希望林默嫁人、过正常生活。机上救亲后，林默将哥哥的死归咎于母亲将她摇醒，母女关系决裂。林默一度想放弃神通、顺应世俗。后续剧情中母亲逐渐理解林默的选择，在焚屋引航时母亲站在灰烬前——是她人生中第一次没有拦她。飞升前母女和解，成为全剧最重的情感落点。

不设爱情线

0621会议明确：以亲情线为核心，不设爱情线。墨娘成仙是自我升华，不因任何人——只因她选择了大爱。

反派 · 海燕公

贯穿全剧的反派。不是纯粹的恶——是人世间坏想法的具象化，撺掇别人干坏事。前期为所有事件（包括父亲和哥哥的死）埋下伏笔。最后被妈祖感化，重新守护海洋。背后可能设置更大的阴谋，为后续二三四季铺垫。
角色形象 · 3D仿真人+CG渲染
统一风格：3D仿真人 + CG渲染。统一提示词确保风格一致。细节有松有弛，氛围感为第一要求。皮肤纹理真实，允许略有瑕疵。避免明星脸版权问题，可结合多个明星特征设计。先出氛围海报图 → 反推角色形象 → 修正服饰形态。

林默（少女妈祖）
女 · 16-28岁 · 主角
朝气明媚，飒爽英气。眼神灵动且带怜悯，但不过于悲悯。服装偏中性，劲装搭配马甲外挂，便于活动。有完整的成长弧线——从活泼冲动到逐渐沉稳。
✨
妈祖（成仙后）
女 · 神格 · 主角
成熟稳重。法天象地——周身云雾、龙游其间、雷电交加，脸部保持少女的宁静。服装极尽华丽：九龙凤冠、披风、飘带。单独制作头冠/衣服后穿到角色身上。

六日上人

皮肤较黑，佩戴生锈且用白布缠着的眼镜。服装根据朝代更换，MV中穿小兵衣服。亦庄亦谐的庙公，妈祖在人间的代言人。

MV预告片 · 内容框架
开场
海浪打船身，柱子吱嘎响。郑和持圣杯跪地祈祷。武将同跪。中央妈祖像。六日上人穿水兵服举签筒。
高潮
签筒内签剧烈晃动 → 一支签化金光冲破云层 → 打回海面 → 海水涨潮 → 妈祖法相空中现身。
画面策略
局部增强张力。神技法天象地。不拍全景只拍局部——手、眼、衣角。大部分展现神仙神技。
